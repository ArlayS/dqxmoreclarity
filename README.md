# DQX More Clarity

A translation utility for the game "Dragon Quest X".

Based on original dqxclarity source code

See the [wiki](https://dqx-translation-project.github.io/dqxclarity/) for how to get started!

## Disclaimer

**I forfeit any responsibility if you receive a warning, a ban or a stern talking to while using this program. `DQX More Clarity` alters process memory, but only for the intent of allowing English-speaking players to read the game in their language. No malicious activities are being performed with `DQX More Clarity`. The goal of this program is simply to translate the game for non-Japanese speaking players to enjoy.**
